# Documentação do repositório

Este repositório reúne diferentes exercícios em Python, realizados no Visual Studio Code, com foco em lógica de programação e manipulação básica de dados.

## Objetivo

Praticar e consolidar conceitos básicos de programação em Python: tomadas de decisão, laços, funções simples e entrada/saída de dados.

## Conteúdo

1. **Contador de Números Pares e Ímpares** – arquivo `Contador de Números Pares e Ímpares.py`
   Exercício de lógica que conta ou classifica números como par ou ímpar.
2. **Cálculo de Média Escolar** – arquivo `Cálculo de Média Escolar.py`
   Programa que calcula média de notas e avalia aprovação ou reprovação.
3. **Jogo de Adivinhação** – arquivo `Jogo de Adivinhação.py`
   Um jogo interativo em que o usuário tenta adivinhar um número escolhido pelo computador.
4. **Tabuada Personalizada** – arquivo `Tabuada Personalizada.py`
   Gera a tabuada de um número informado pelo usuário, dentro de um intervalo.
5. **Validador de Senha Simples** – arquivo `Validador de Senha Simples.py`
   Verifica se a senha digitada atende a critérios básicos (tamanho, caracteres, etc).

## Dificuldades comuns e dicas

* Validar a entrada do usuário (por exemplo, garantir que seja um número, evitar erros de digitação).
* Verificar e tratar casos extremos (como zero, negativo, entradas inválidas).
